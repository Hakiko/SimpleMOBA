# SimpleMOBA

## Requirements
- Qt >= 5.6
- C++ compiler with C++11 support
- OpenGL 2.0 ES (libgl1-mesa-dev)

## Install
```
git clone --recursive https://github.com/io-2016/SimpleMOBA.git
```
